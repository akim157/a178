JavaScript

JavaScript - это клиентский язык на котором пишут различные сценарии, скрипты. Он является важным для изучения каждого веб-мастера, так как раскрывает большие возможности для дизайна сайта.

При объединении HTML, CSS, и JavaScript появляется DHTML(Dynamic HTML), оно позволяет реализовать любой дизайн сайта.

Кроме внешнего вида JavaScript может провести проверку корректности данных формы, работать с действиями пользователя, таким как нажатие клавиш, движение мышки и т.д.

Действия JavaScript неограничены все это и больше вы можете узнать в бесплатных уроках по JavaScript! :)
