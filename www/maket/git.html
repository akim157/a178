GIT

GIT - это система контроля версий. Очень мощная и профессиональная система. Проект Git был создан в 2005 году Линусом Торвальдсом (Для тех кто не знает - Линус - программист, хакер - создатель ядра ОС Линукс).

Системы контроля версий используются всеми разработчиками, веб-компаниями. И на текущий момент лидируют всего три системы.

В подавляющем большинстве случаев в фриланс-проектах одно из требований знать либо Git либо Mercurial. Тот программист который Уже знает - имеет больше шансов получить заказ.
